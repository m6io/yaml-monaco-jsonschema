## Usage

### To install

```bash
yarn
```

### Run in development mode

```bash
yarn dev
```

## Features

- React + Typescript + Vite
- Tailwind CSS
- Monaco Editor
- JSON Schema validation
- Dark mode
- Tailwind CSS class sorting, merging and linting.
- 100% customizable
